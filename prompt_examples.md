# Prompt 示例

## 针对 example_dataset.csv 的原始 Prompt

这个数据集是一个文本分类任务，需要根据标题（Headline）预测 Typography（0 或 1）。

### 推荐的原始 Prompt（简单版本）

```
你是一个文本分类助手。请根据给定的标题，判断其 Typography 类别。

任务：根据标题内容，预测 Typography 是 0 还是 1。

只返回数字 0 或 1，不要包含其他文字。
```

### 推荐的原始 Prompt（详细版本）

```
你是一个文本分类专家。你的任务是根据新闻标题预测其 Typography 类别。

输入：一个新闻标题
输出：0 或 1（只返回数字，不要包含其他文字）

请仔细分析标题的内容、风格和特征，然后给出分类结果。
```

### 推荐的原始 Prompt（带示例版本）

```
你是一个文本分类助手。根据标题预测 Typography 类别。

规则：
- 输入：新闻标题
- 输出：只返回 0 或 1（数字）

示例：
标题："Daily life in RAJASTHAN, INDIA 🇮🇳- CITY vs RURAL"
输出：1

标题："10 of the Coolest Cars Ever Made & 10 That Nobody Cares About"
输出：0

现在请对以下标题进行分类，只返回 0 或 1：
```

### 推荐的原始 Prompt（分析型版本）

```
分析给定的新闻标题，判断其 Typography 类别。

Typography 是一个二分类标签（0 或 1），需要根据标题的特征来判断。

请仔细分析标题的以下特征：
- 标题的长度和结构
- 使用的词汇和表达方式
- 标题的风格和语气

输出要求：只返回数字 0 或 1，不要包含任何其他文字或解释。
```

## 使用建议

1. **从简单版本开始**：先用最简单的 prompt，让系统自动优化
2. **观察优化过程**：查看每轮迭代的改进建议，了解系统如何优化 prompt
3. **根据结果调整**：如果优化后的 prompt 不理想，可以尝试从详细版本开始

## 数据集说明

- **输入字段**：Headline（标题）
- **输出字段**：Typography（0 或 1）
- **任务类型**：二分类

系统会自动识别 Headline 作为输入，Typography 作为 ground_truth。
